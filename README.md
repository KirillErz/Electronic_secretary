# Flask + Svelte Auth

Session-based auth, decoupled front and backend, cross origin

## Getting Started

Run Flask:

```sh
$ cd backend
$ python3.9 -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt
$ python app.py
```

Run Svelte:

```sh
$ cd frontend
$ npm install
$ npm run dev
```

Test at [http://localhost:8080/](http://localhost:8080/)
